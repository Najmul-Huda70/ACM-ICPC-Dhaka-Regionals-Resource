# 2.0 Basic Math and Again Number Theory
## Divisibily and Large Number
how to check if a number a<=10^100000 is divisible by b<=10^9 ? 

input :
```cpp
string a; 
int b; 
cin >> a >> b;
```

This allows a to be extremely large — even 10⁵ or 10⁶ digits.

Then you compute a % b manually:
```cpp
bool isDivisible(string a, int b)
{
    int ans = 0;
    for (auto c : a)
    {
        ans = (ans * 10LL % b + c - '0') % b;
    }
    return ans == 0;
}
```

This uses the property:
```cpp
( (x * 10) + d ) % b = ( ((x % b) * 10 % b) + d ) % b
```

So you never overflow.

Then:
```cpp
if (ans == 0) 
    cout << "a is divisible by b\n";
else 
    cout << "sad\n";
```
